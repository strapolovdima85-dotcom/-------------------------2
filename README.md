Анализ финансовой отчётности российских компаний (RFSD)

Проект представляет собой экспресс-анализ датасета RFSD «Финансовая отчётность российских компаний» 

Описание работы
В ходе исследования были выполнены следующие этапы:
1. Загрузка данных: 
    - Использование режима streaming
2. Предобработка:
    - Выборка ограничена 1000 строками и 20 колонками.
    - Выполнена локализация названий показателей (перевод кодов ОКУД в понятные финансовые термины: Выручка, Активы и т.д.).
    - Идентификаторы (ИНН, Название компаний) исключены из статистического анализа для чистоты результатов.
3. Анализ:
    - Построены гистограммы распределения, рассчитана матрица корреляции и диаграмма рассеивания.

Ключевые выводы:
    - Кратный разрыв между медианой и максимальными значениями показателях Выручки говорит о том, что средние значения смещены из-за нескольких гигантов.
    - Облако точек на графике «Выручка vs Себестоимость» вытянуто в узкую прямую линию. Выпадающие значения являются аномалиями, это могло быть при сверхэффективности или налоговых манипуляциях.

Хранение данных
